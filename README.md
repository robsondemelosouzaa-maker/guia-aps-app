# Guia APS App - Fase 1 (UX/UI)

Este projeto foi construído focando nos princípios de **Design Human-Centered** e na redução de carga cognitiva para profissionais de enfermagem na Atenção Primária à Saúde.

## 🚀 Como Executar
1. Certifique-se de ter o **Node.js** instalado na sua máquina.
2. Na pasta do projeto, execute:
   ```bash
   npm install
   npm run dev
   ```

## 🎨 Princípios de Design
- **Arquitetura de Escolha**: Prioriza o próximo passo mais provável.
- **Microcopy**: Textos curtos, diretos e empáticos.
- **Glassmorphism**: Estética premium e moderna.
- **Responsivo**: Foco total no uso em dispositivos móveis (Mobile-First).

## 🛠 Features Implementadas (Mock)
- **Dashboard**: Visão geral de pendências e métricas.
- **CIAP-2**: Busca rápida com zero atrito.
- **Módulos**: Navegação amigável por áreas de saúde.
- **Configurações**: Acessibilidade e Modo Demonstração.
- **Guia de Estilo**: Visualização de componentes em `/ui`.
